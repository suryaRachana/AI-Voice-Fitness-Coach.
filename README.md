# 🏋️ AI Voice Fitness Coach

**A web-based AI-powered fitness coach that gives real-time voice instructions and countdown timers for workouts.**

## Features
- 🎙️ Voice instructions for each exercise
- ⏱️ Countdown timers for workouts & rests
- 🔄 Sequenced exercises for smooth flow
- 🧩 Easy to extend with different workout plans
- 💻 Simple web-based interface

## How It Works
1. User clicks **Start Workout**.
2. The app guides exercises **step-by-step**.
3. Voice instructions are read aloud using **Text-to-Speech (TTS)**.
4. Countdown timer shows time left for each exercise.
5. Workout completes with a **final message** congratulating the user.

## Technology Used
- HTML, CSS, JavaScript
- Browser **SpeechSynthesis API** for TTS

## Future Enhancements

- Custom workout plans and difficulty levels
- Progress tracking and visual progress bar
